# 🌊 Smart Marine Fleet Telemetry & Management System (SaaS)

An end-to-end IoT platform designed for **real-time tracking**, **remote control**, and **keyless rental management** of marine vehicles.

---

## 🚀 Overview

This system enables:

- Real-time vehicle tracking
- NFC-based keyless access (Apple Wallet / Google Wallet)
- Remote engine control
- Theft prevention
- Multi-tenant SaaS platform

---

## 🧱 Architecture

[ IoT Device ] ↓ MQTT (TLS) 
[ Broker ] ↓ 
[ Backend ] ↓ 
[ Database ] ↓ 
[ Web Panel ]
